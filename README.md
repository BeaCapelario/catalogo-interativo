Este repositório contém o desenvolvimento de um catálogo interativo de produtos ou itens. O projeto foi focado na criação de uma interface dinâmica onde o usuário pode explorar elementos de forma fluida, aplicando conceitos de filtragem, busca e exibição de detalhes em tempo real.

---

### O que foi explorado:
* Interatividade em Tempo Real: Atualização da interface baseada nas ações e seleções do usuário.
* Manipulação Dinâmica de Conteúdo: Renderização de elementos a partir de estruturas de dados JavaScript.
* Sistemas de Filtro: Lógica para categorização e seleção específica de itens do catálogo.
* Design de Interface (UI): Criação de layouts organizados para facilitar a visualização de grandes volumes de informação.
* UX (Experiência do Usuário): Foco em transições e feedback visual durante a navegação entre os itens.
